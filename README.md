# Фінансовий трекер студента (Консольний чат-бот)

## Мета проєкту
Розробити консольного бота мовою Python, який допомагає студенту вести облік особистих витрат, контролювати встановлений бюджет, а також аналізувати витрати за допомогою фільтрів та звітів.
## Опис логіки та алгоритму роботи бота

Програма працює за принципом нескінченного циклу (while), очікуючи на введення команд від користувача. 
Алгоритм роботи:
**Ініціалізація:** При запуску програма намагається зчитати дані з файлу data.json. Якщо файл відсутній, створюється початкова структура.
**Введення:** Користувач вводить команду.
**Обробка:** Бот перевіряє команду через умовні конструкції (if/elif/else) та викликає відповідну функцію.
**Збереження:** При додаванні нових витрат або зміні бюджету дані автоматично перезаписуються у файл data.json.
**Валідація:** Бот перевіряє, чи не перевищено ліміт бюджету під час додавання витрати, а також перевіряє правильність вводу даних (числа, дати).
## Опис структури даних

Дані зберігаються у форматі json у вигляді словника.
budget (float) — загальна сума бюджету.
expenses (list) — список словників, де кожен словник представляє окрему витрату.

**Структура JSON-файлу:**
```json
{
    "budget": 5000.0,
    "expenses": [
        {
            "amount": 150.0,
            "category": "Їжа",
            "date": "11-06-2026",
            "comment": "Обід у їдальні"
        }
    ]
}
